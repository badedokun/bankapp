OrokiiPay

Multi-Tenant Banking Platform - Registration Flow Mockups

Complete User Registration & KYC System

This demonstration showcases a comprehensive user onboarding system with Know Your Customer (KYC) compliance for multi-tenant banking applications.

Backend API: http://localhost:3001/api/registration/

Current Tenant: First Midas Microfinance Bank (fmfb)

Registration Flow

1
Account Creation
Initial registration with personal information, email, and phone number verification setup.
  • Personal information collection (name, date of birth)
  • Email and phone number validation
  • Strong password requirements
  • Terms and conditions acceptance
  • Real-time form validation
View Page
2
Contact Verification
Two-factor verification using email tokens and SMS codes to confirm user identity.
  • Email verification with secure tokens
  • SMS verification with 6-digit codes
  • Resend functionality for failed attempts
  • Progress tracking and visual feedback
  • Session management with localStorage
View Page
3
Personal Details
Comprehensive profile completion with address, employment, and Nigerian banking identifiers.
  • Complete address information with Nigerian states
  • Employment and income details
  • BVN (Bank Verification Number) integration
  • NIN (National Identity Number) support
  • Real-time validation for Nigerian banking standards
View Page
4
KYC Documents
Secure document upload system for regulatory compliance with drag-and-drop functionality.
  • Government ID document upload
  • Proof of address verification
  • Selfie with ID for biometric matching
  • Signature sample collection
  • Drag-and-drop file uploads with validation
View Page
5
Completion & Success
Registration confirmation with next steps, account status, and onboarding guidance.
  • Registration success confirmation
  • Account status overview
  • KYC processing timeline
  • Next steps and guidance
  • Support information and contacts

API Endpoints

These mockups integrate with the following RESTful API endpoints:

POST /api/registration/start
POST /api/registration/verify-email
POST /api/registration/verify-phone
POST /api/registration/personal-details
GET /api/registration/status/:userId

Technology Stack

Frontend Vanilla HTML/CSS/JavaScript with modern ES6+ features
Backend Node.js, Express.js, TypeScript
Database PostgreSQL with tenant schema isolation
Security bcrypt, JWT, rate limiting, helmet.js
Validation express-validator with Nigerian banking compliance
Multi-tenancy Tenant middleware with X-Tenant-ID headers
