Integration Guide

How to use the design system in your projects.

HTML/CSS Usage

Include the CSS file and use the component classes:

<link rel="stylesheet" href="orokii-pay-theme.css"> <button class="orokii-btn orokii-btn--md orokii-btn--primary"> Primary Button </button> <div class="orokii-card orokii-card--md"> <div class="orokii-card__header"> <div class="orokii-card__title">Card Title</div> </div> <p>Card content goes here.</p> </div>

React Native Usage

Import and use the design system functions:

import { createButtonStyles, defaultThemes } from '@/design-system'; const theme = defaultThemes.orokiiPay; const buttonStyles = createButtonStyles(theme, { variant: 'primary', size: 'md' }); <TouchableOpacity style={buttonStyles.container}> <Text style={buttonStyles.text}>Primary Button</Text> </TouchableOpacity>
